This plugin automatically updates language files when new plugin versions contains newer language files.

When using own customized phrases place them in new files with the same layout and set the 'override' setting to true.
Remember -> the customized files must start with the language tag like en_us or de_DE (ex.: de_DE.plain.override.xml)
Only place customized phrases inside the file. The system will use ALL phrases inside your file and ignore the phrases included inside the plugin
